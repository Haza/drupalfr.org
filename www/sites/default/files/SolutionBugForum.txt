COMMENT REMETTRE EN MARCHE LE MODULE FORUM APRÈS UNE ACTIVATION/DÉSACTIVATION/ACTIVATION DE CELUI-CI.Il faut “reconnecter” le vocabulaire “Forum” avec les termes associés aux containers et aux forums existants.Pour cela:	- créer ou mettre à jour le vocabulaire “Forum”, dans admin/content/taxonomy, en:		— cochant “Sujet de discussion” dans “Types”		— cochant “Simple” dans “Hiérarchie”		— cochant “Obligatoire”		— mettant le poids à: -10			- sous phpMyAdmin:		— dans la table “term_data”, noter la valeur du champ “vid” dans les lignes correspondant aux termes définissant les forums et les containers présents sur le site			Notons val cette valeur.		— dans la table “vocabulary”, dans la ligne correspondant à “name==Forum” mettre “vid=val”,		— dans la table “vocabulary_node_types”,  dans la ligne correspondant à “name==forum”, mettre “vid=val”.		Et, normalement, ça marche!
